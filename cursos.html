Carga Horária

O treinamento tem 20 horas de duração, sendo dividido em 5 aulas de cerca de 4 horas.


Resultados Esperados

  • Entender o funcionamento do Docker
  • Entender, criar e gerenciar Containers
  • Entender, criar e gerenciar imagens
  • Subir suas imagens para registries públicos e privados na internet
  • Criar seu próprio registry privado
  • Visão geral das ferramentas mais utilizadas para gerenciar o Docker e suas aplicações na operação de sistemas
  • Realizar provisionamento de máquinas virtuais em cluster swarm
  • Realizar Deploy de Aplicações em um Cluster Local, Amazon Web Service (AWS) e Digital Ocean
  • Criar Cluster geograficamente distribuído
  • Criar redes conectando data centers entre diferentes Cloud Providers

Público-Alvo

  • Desenvolvedores de software

  • Pré-Requisitos

    • Levar o seu próprio notebook.
    • Preparar préviamente seu ambiente com as ferramentas que serão utilizadas (Forneceremos um vídeo tutorial com as instruções para proceder com a instalação).
    • Ter conhecimento de Fundamentos de redes de computadores, sistemas operacionais, experiência com alguma distribuição Linux e noções de virtualização.

    Mecânica

    • As aulas misturam conteúdo teórico com atividades práticas onde os alunos vão construindo passo-a-passo o ambiente para o deploy de uma aplicação em cluster. Durante essa jornada, discutiremos a estratégias e melhores práticas para utilizar as ferramentas e os pontos fortes dessa tecnologia em nosso benefício.
